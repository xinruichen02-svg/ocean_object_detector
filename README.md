# Underwater-Robot-Object-Detection

面向水下机器人作业的水下目标识别算法学习与工程实践。

这个仓库目前是一个学习型科研工程仓库，用来系统学习水下目标检测、整理实验过程、记录问题分析，并逐步搭建面向水下机器人视觉感知的工程结构。

## 项目简介

本项目围绕水下机器人视觉识别任务展开，重点关注水下图像中的目标检测、数据集整理、YOLO baseline、水下图像增强、失败案例分析，以及后续可能连接到机器人感知接口的工程实践。

## 为什么做水下目标识别

水下机器人在海洋观测、目标巡检、垃圾清理、水产养殖、设备维护等任务中，需要通过视觉系统识别周围环境和作业目标。目标识别能力越稳定，机器人后续的定位、跟踪、路径规划和机械臂操作就越容易建立。

水下环境比普通陆地场景更复杂，因此这个方向既适合学习目标检测基础，也适合进一步理解真实工程中数据质量、环境退化和系统接口带来的问题。

## 水下目标识别和普通目标检测的区别

普通目标检测通常假设图像清晰度、光照和颜色分布相对稳定；水下目标检测则经常面对以下问题：

- 光照不足，远距离目标容易变暗。
- 水体吸收和散射导致图像偏蓝或偏绿。
- 浑浊水体会降低对比度，使目标边界不清楚。
- 小目标较多，目标尺度变化明显。
- 背景复杂，珊瑚、石块、海草、悬浮物可能造成误检。
- 视频中机器人运动和水流会带来抖动，影响检测稳定性。

因此，本项目不仅学习检测模型本身，也会逐步关注数据质量、图像增强、失败案例和机器人接口。
