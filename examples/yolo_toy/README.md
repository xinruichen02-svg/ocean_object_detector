# YOLO 玩具练习

这个目录完整保存最初用于跑通 YOLO 流程的教学示例。它与正式水下项目隔离，
只能证明数据读取、训练、验证和预测链路能够运行，不能说明模型识别效果。

```text
dataset/               4 张合成图片及 YOLO 标签
experiment_notes.md    当时的命令、环境处理和观察记录
outputs/               本地训练与预测输出，不提交 Git
weights/               玩具练习使用的权重，不提交 Git
```

所有示例命令都从仓库根目录执行。正式项目入口位于
`projects/underwater_static_object_detection/README.md`。
