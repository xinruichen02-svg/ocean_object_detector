# 通用学习区

这里保存可以跨项目复用的目标检测知识、学习路线和个人问题，不保存正式数据、
模型权重或实验结果。

```text
learning_plan.md       目标检测到机器人视觉节点的学习路线
yolo_format_notes.md   YOLO 数据格式学习笔记
dataset_notes.md       水下数据集通用记录模板
questions.md           学习过程中尚未解决的问题
```

学会并确认的内容，再落实到 `projects/underwater_static_object_detection/`；
需要动手验证的小实验放到 `examples/`。
