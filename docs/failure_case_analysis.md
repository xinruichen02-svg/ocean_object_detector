# 失败案例分析模板

失败案例分析用于帮助我理解模型为什么出错。水下目标检测中，坏例子往往比好结果更能说明问题。

## 常见失败类型

- 小目标漏检
- 低光照漏检
- 浑浊场景漏检
- 偏蓝/偏绿导致误检
- 遮挡导致漏检
- 背景复杂导致误检
- 视频抖动导致识别不稳定
- 相似类别混淆

- 是否需要以后重点分析：

## 分析提醒

- 先区分漏检、误检和类别混淆。
- 再观察是否和水下环境有关，例如低光照、偏色、浑浊、遮挡。
- 如果是小目标问题，要记录目标大概尺寸和背景情况。
- 如果是视频问题，要记录是否存在抖动、模糊或连续帧不稳定。
